News Dashboard

React application that uses the NewsAPI to retrieve US news. Users can search for a news topic and see the top 20 search results for that category. Additionally, users will see a data chart summarizing where the news is coming from. Once a search is made, the app tallies each news source and displays the data on a bar chart. Unfortunately, NewsAPI only allows the free tier of their API usage to run locally, so there is no live version of the project. :(
